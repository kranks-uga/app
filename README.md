# Альфонс

Помощник для Arch Linux с AI интеграцией через Ollama.

## Возможности

- **AI ассистент** — отвечает на вопросы, использует локальную модель через Ollama
- **Управление пакетами** — поиск, установка, удаление через yay/pacman
- **Обучающие гайды** — пошаговые инструкции по настройке системы
- **Системные команды** — время, дата, выключение, перезагрузка

## Требования

- Arch Linux
- Rust 1.70+
- [Ollama](https://ollama.ai/) с моделью llama3

## Установка

```bash
# Клонировать репозиторий
git clone https://github.com/kranks-uga/app.git
cd app

# Установить rust
sudo pacman -S rustup
rustup default stable

# Установить Ollama и загрузить модель
yay -S ollama
ollama pull llama3

# Запустить Ollama
ollama serve

# Собрать и запустить
cargo run --release
```

## Команды

### Базовые
| Команда | Описание |
|---------|----------|
| `время` | Текущее время |
| `дата` | Текущая дата |
| `помощь` | Список команд |
| `очистить` | Очистить чат |

### Пакеты
| Команда | Описание |
|---------|----------|
| `найти <пакет>` | Поиск пакетов |
| `установить <пакет>` | Установка пакета |
| `удалить <пакет>` | Удаление пакета |
| `обновить систему` | Обновление всех пакетов |

### Гайды
| Команда | Описание |
|---------|----------|
| `гайд pacman` | Основы pacman |
| `гайд aur` | Работа с AUR |
| `гайд wifi` | Настройка WiFi |
| `гайд systemd` | Управление сервисами |
| `гайд gpu` | Установка драйверов |
| `гайд audio` | Настройка звука |
| `гайд locale` | Локализация |
| `гайд backup` | Резервное копирование |

### Система
| Команда | Описание |
|---------|----------|
| `выключить пк` | Выключение компьютера |
| `перезагрузить` | Перезагрузка |

## Структура проекта

```
src/
├── main.rs              # Точка входа
└── app/
    ├── mod.rs           # Модуль приложения
    ├── assistant_app.rs # Главная структура
    ├── chat.rs          # Чат и фоновые задачи
    ├── config.rs        # Настройки
    ├── constants.rs     # Константы
    ├── ai/
    │   ├── local_provider.rs  # Ollama клиент
    │   └── tools.rs           # Инструменты AI
    ├── commands/
    │   ├── base.rs      # Базовые команды
    │   ├── package.rs   # Пакетный менеджер
    │   ├── guide.rs     # Гайды
    │   └── system.rs    # Системные команды
    ├── guides/
    │   └── mod.rs       # Реестр гайдов
    └── ui/
        ├── mod.rs       # Главный UI
        ├── widgets.rs   # Виджеты
        └── dialogs.rs   # Диалоги
```

## Настройки

Настройки хранятся в `~/.config/alfons-assistant/config.json`:
- Имя ассистента
- Цвет темы
